LoadOptimize

Designed a weighted optimization engine using 4 scoring parameters (utilization, route deviation, cost, CO2). Generated Top-3 to Top-10 ranked truck matches per shipment request. Reduced empty runs by prioritizing >70% capacity utilization and sustainable routing.

DPR Analyser

Developed an AI pipeline evaluating DPRs across 3 dimensions: compliance, financial risk, and feasibility. Transformed 100+ page DPR PDFs into structured schemas using LLM-based extraction and RAG. Enabled ranked comparison and approval workflows for administrators via AI-generated justifications.

GiftAI

Engineered an AI chatbot generating recommendations from 4 inputs (interests, occasion, budget, personality). Integrated real-time product retrieval with images, prices, and reviews via external APIs. Reduced gifting decision time by replacing manual search with conversational AI.

SmartCampus

Designed a database schema for student enrollments and conflict-free timetables. Implemented Spring Security RBAC for 115+ REST endpoints. Built a scalable notification and attendance management system.

SmartClinic

Architected a full-stack platform supporting 4 user roles (patients, doctors, staff, admins). Implemented GPS-based staff check-ins with 150–200m radius validation. Automated queues, reminders, and records to reduce clinic coordination delays.
